# AuStepWizard 步骤组件

### 简介

步骤向导组件，支持步骤导航、步骤切换和自定义按钮配置。将复杂业务流程拆分为多个步骤引导用户完成，提供完成状态展示，适用于订单创建、审批流程等场景。
