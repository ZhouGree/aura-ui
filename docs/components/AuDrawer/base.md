# AuDrawer 抽屉组件

### 简介

抽屉弹窗组件，支持异步确认和多方向弹出。提供编程式控制方法，异步确认功能确保操作完成后再关闭弹窗，适用于表单编辑、详情查看等侧边栏场景。
